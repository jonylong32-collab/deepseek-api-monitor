# Code of Conduct

This project welcomes practical, respectful collaboration.

## Expected Behavior

- Be respectful and constructive.
- Keep issues and pull requests focused on the project.
- Avoid sharing private credentials, account data, or personal information.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Spam or unrelated self-promotion.
- Publishing secrets, tokens, or private account data.

Maintainers may remove comments, issues, or pull requests that do not follow this code.
